# Sitemap Configuration

This document explains how the sitemap is configured for SEO and search engine indexing.

## Overview

The sitemap is automatically generated by Next.js from `app/sitemap.ts` and is accessible at:
- **Sitemap URL**: `https://kupennia.co.zw/sitemap.xml`
- **Robots.txt**: `https://kupennia.co.zw/robots.txt`

## What's Included

The sitemap includes all pages of the website:

### 1. Static Pages (Priority: 0.8-1.0)
- Homepage (`/`) - Priority: 1.0, Weekly updates
- Services Listing (`/services`) - Priority: 0.9, Monthly updates
- Cleaning Services (`/cleaning-services`) - Priority: 0.8, Weekly updates
- About Page (`/about`) - Priority: 0.8, Monthly updates
- Contact Page (`/contact`) - Priority: 0.8, Monthly updates

### 2. Service Pages (Priority: 0.8)
All individual service pages:
- `/services/bnb-cleaning`
- `/services/deep-cleaning`
- `/services/domestic-cleaning`
- `/services/commercial-cleaning`
- `/services/office-cleaning`
- `/services/carpet-cleaning`
- `/services/window-cleaning`
- `/services/mattress-cleaning`
- `/services/couch-cleaning`
- `/services/car-seat-cleaning`
- `/services/car-washing`
- `/services/pressure-washing`
- `/services/bin-cleaning`
- `/services/apartment-complex-cleaning`
- And all other services...

**Update Frequency**: Weekly
**Priority**: 0.8

### 3. Service Area Combination Pages (Priority: 0.6)
All combinations of services and areas:
- `/services/bnb-cleaning/tynwald`
- `/services/deep-cleaning/borrowdale`
- `/services/domestic-cleaning/glen-lorne`
- And all other service × area combinations...

**Update Frequency**: Monthly
**Priority**: 0.6

## How It Works

### Automatic Generation
Next.js automatically:
1. Reads `app/sitemap.ts` during build time
2. Generates XML sitemap at `/sitemap.xml`
3. Updates it whenever you rebuild the site

### Dynamic Content
The sitemap dynamically includes:
- All services from `lib/serviceData.ts`
- All service areas from `lib/areaData.ts`
- All combinations of services × areas

### Search Engine Submission

#### Google Search Console
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property: `https://kupennia.co.zw`
3. Go to **Sitemaps** section
4. Submit: `https://kupennia.co.zw/sitemap.xml`

#### Bing Webmaster Tools
1. Go to [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. Add your site
3. Submit sitemap: `https://kupennia.co.zw/sitemap.xml`

#### Automatic Discovery
Search engines automatically discover your sitemap through:
- `robots.txt` file (references the sitemap)
- Sitemap location header (if configured on server)

## Robots.txt Configuration

The `app/robots.ts` file generates `robots.txt` with:
- **Allow**: All pages (`/`)
- **Disallow**: 
  - `/api/` - API endpoints (not for indexing)
  - `/admin/` - Admin panel (not for indexing)
- **Sitemap**: `https://kupennia.co.zw/sitemap.xml`

## SEO Benefits

1. **Complete Coverage**: All pages are included in the sitemap
2. **Priority Signals**: Important pages have higher priority
3. **Update Frequency**: Tells search engines how often to re-crawl
4. **Last Modified Dates**: Helps search engines know when content changed

## Updating the Sitemap

### Adding New Services
1. Add service to `lib/serviceData.ts`
2. Sitemap automatically includes it on next build

### Adding New Areas
1. Add area to `lib/areaData.ts`
2. Sitemap automatically includes all service × area combinations

### Changing Priorities
Edit `app/sitemap.ts` to adjust:
- `priority` values (0.0 to 1.0)
- `changeFrequency` values ('always', 'hourly', 'daily', 'weekly', 'monthly', 'yearly', 'never')

## Best Practices

✅ **Do:**
- Keep sitemap updated with all pages
- Use appropriate priorities
- Set realistic update frequencies
- Submit to search engines
- Monitor in Search Console

❌ **Don't:**
- Include duplicate URLs
- Include pages blocked by robots.txt
- Set all priorities to 1.0
- Include admin or API pages
- Forget to update after adding new content

## Troubleshooting

### Sitemap Not Updating
- Rebuild the site: `npm run build`
- Clear Next.js cache: Delete `.next` folder
- Check for build errors

### Search Engines Not Finding Sitemap
- Verify `robots.txt` references sitemap
- Submit manually in Search Console
- Check sitemap is accessible: `curl https://kupennia.co.zw/sitemap.xml`

### Too Many URLs
- Google limit: 50,000 URLs per sitemap
- Current: ~3,687 URLs (well under limit)
- If needed, split into multiple sitemaps using sitemap index
