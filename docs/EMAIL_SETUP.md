# Email Notification Setup Guide

## Free SMTP Services Comparison

### 1. **Resend** (Recommended for Next.js) ⭐
- **Free Tier**: 3,000 emails/month, 100 emails/day
- **Best For**: Modern Next.js applications
- **Pros**: 
  - Built specifically for developers
  - Easy Next.js integration
  - Great API and documentation
  - No credit card required
- **Sign Up**: https://resend.com

### 2. **Brevo (formerly Sendinblue)**
- **Free Tier**: 300 emails/day (9,000/month)
- **Best For**: Marketing + transactional emails
- **Pros**: 
  - Higher free limit
  - Includes CRM features
  - Good deliverability
- **Sign Up**: https://www.brevo.com

### 3. **SendGrid**
- **Free Tier**: 100 emails/day (forever free)
- **Best For**: Reliable transactional emails
- **Pros**: 
  - Very reliable
  - Good documentation
  - Industry standard
- **Sign Up**: https://sendgrid.com

### 4. **Mailgun**
- **Free Tier**: 5,000 emails/month (first 3 months), then 1,000/month
- **Best For**: Developers who need API flexibility
- **Pros**: 
  - Developer-friendly API
  - Good for transactional emails
- **Sign Up**: https://www.mailgun.com

### 5. **Amazon SES**
- **Free Tier**: 62,000 emails/month (when sent from EC2)
- **Best For**: High volume, AWS users
- **Pros**: 
  - Very high free limit
  - Pay-as-you-go pricing
- **Cons**: Requires AWS account setup

## Recommendation: **Resend**

Resend is the best choice for Next.js projects because:
- ✅ Designed for modern web apps
- ✅ Simple API
- ✅ Great TypeScript support
- ✅ Easy domain verification
- ✅ No credit card required for free tier
